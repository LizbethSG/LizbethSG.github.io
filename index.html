METADATA

El elemento head alberga la mayor parte de la información leída por máquina en un documento HTML. Veremos algunos elementos de metadatos de uso común y cómo mejoran una página web y cómo pueden optimizar una página web para motores de búsqueda y navegadores modernos.

• Los siguientes elementos se consideran contenido de metadatos: base, link, meta, noscript, script, style y title
